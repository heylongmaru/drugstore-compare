# 日本藥妝比價（drugstore-compare）

路過日本藥妝店看到想買的，隨手記一筆「商品＠店 ¥價格」；清單並排各店標價、標出最低價。要結帳前開「結算」，自動套各店優惠券級距、免稅 10%、匯率與付款回饋，算出「整車去哪家結最划算」。

- **線上使用**：開啟 GitHub Pages 網址即可，手機可「加入主畫面」當 App 用
- 單檔 HTML、離線可用（匯率抓取需網路，斷網沿用上次值）
- 資料只存在裝置瀏覽器（localStorage），不上傳；換裝置用「設定 → 匯出／匯入 JSON」
- 內建大國藥妝、松本清、LOFT、唐吉訶德等常見店；優惠級距為 2026-07 參考值，**以現場公告為準**，可自行修改
- 2026/11/1 起日本免稅改「退稅制」：比價邏輯不變，詳見 [guide.html](guide.html)
